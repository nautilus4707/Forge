# Quickstart Example

This example demonstrates a minimal Forge agent configuration.

## Usage

1. Install Forge from the project root: `pip install -e .`
2. Run the agent via YAML: `forge run -f hello_agent.yaml "Hello!"`
3. Run the agent via Python: `python hello_agent.py`

## Customization

Edit `hello_agent.yaml` to change the model, system prompt, or enabled tools.
